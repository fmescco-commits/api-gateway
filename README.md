# API Gateway

Este proyecto corresponde al API Gateway del sistema de comercio electrónico basado en microservicios.

## Tecnologías
- Lenguaje / Framework: (colocar)
- Arquitectura: API Gateway
- Comunicación: HTTP / REST
- Contenedores: Docker

## Descripción
El API Gateway actúa como punto único de entrada al sistema, redirigiendo las solicitudes del cliente hacia los microservicios correspondientes.

## Ejecución
El API Gateway se despliega mediante Docker Compose y permite el acceso unificado a los microservicios de usuarios, productos y pedidos.
